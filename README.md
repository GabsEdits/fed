## Install
```
git clone https://github.com/GabsEdits/fed/ && cd fed
```
```
sudo sh script.sh
```
## Usage
```
    Usage: script.sh [-g|-f|-r|-m|-a|-x]
-g    Do all flags (f,r,m,a)
-f    Make dnf faster
-r    Add RPM Fusion
-m    Add Multimedia Codecs
-a    Install apps
-x    Add Apx *don't run with root*
```
